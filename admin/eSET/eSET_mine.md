# Supplemental eSET Questions

## Content coverage

- Which class topics did you find most and least useful or effective?

- Are there topics you wish had been covered or covered in greater depth?

- Should there be more discussion of AI tools (effective uses, ethics, etc.)?

## Format and structure

- Which class activities did you find most and least useful or effective?

- Should the course include more activities, more writing time, more lecture/discussion classes, or is the current balance about right?

- How might I motivate more discussion in class? (How did you use / engage with the slides? Did you review them before class?)

## Specific course features

- Was pairing up with an accountabilibuddy helpful for your productivity and motivation? Should this be continued in future offerings?

## Feedback and accountability

- Did you receive enough individual feedback on your writing — from the instructor, peers, or both — to make meaningful revisions?

- Would additional accountability requirements (e.g., graded exercises, mandatory manuscript benchmarks) have been a useful motivator for greater productivity?
